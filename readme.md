mkd2
====

Yet another build automation utility.

System requirement
------------------

Win32 or POSIX.

Installation
------------

1. Compile main.c. Any of these works:
	```shell
	gcc main.c -o mkd2
	clang main.c -o mkd2
	cl main.c /Femkd2.exe
	```
2. Move mkd2(.exe) to somewhere in your $PATH (%PATH%).
