mkd2
====

**[WIP]**

Yet another build automation utility.

System requirements
-------------------

Win32 or POSIX.

Installation
------------

1. Compile everything.c. Any of these works:

	```shell
	gcc everything.c -o mkd2
	clang everything.c -o mkd2
	cl everything.c /Femkd2.exe
	```

2. Move mkd2(.exe) to somewhere in your $PATH (%PATH%).
